Voter Registration Map

Purpose

The objective of this map is to display Hispanic voter registration rate for each census block in the city of Fitchburg, Massachusetts.

Core Processes

  • Utilized Model Builder to efficiently find the number of voters for each ethnic group.
  • Conducted a spatial join to relate the X,Y point voter data to the Fitchburg census blocks.
  • Implemented a graduated color scale to highlight census blocks with the highest percentage of Hispanic Voters.
